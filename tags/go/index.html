关于 Hugo

Hugo 是由 spf13 创建的一个静态网站生成器，首页：https://www.gohugo.io/

Hugo 工作流

下面将介绍Hugo的完整工作流程：

在 https://www.gohugo.io/ 能够找到 Homebrew 安装 Hugo的命令，

下载对应的操作系统版本的Hugo二进制文件（hugo或者hugo.exe），添加到$PATH能找到的目录下。
